Recipe notes


- Babel

./node_modules/.bin/babel index.js -o ./bundle.js --presets=es2015
- [bable path (local)] [target] [write new file] [file name] [preset config (needs independent instalation)]

.babelrc -> File that works as a configuration for compiler

{
  "presets": [
    "es2015"
  ]
}

- Webpack

 - loaders -> applied transformations during bundling
 - Plugins -> applied transformations to output

webpack.config.js ->

const webpack = require('webpack');

module.exports = {
  entry: './index.js',
  mode: 'development',
  output: {
    path: __dirname,
    filename: './bundle.js'
  },
  module: {
    rules: [{
      test: /\.js$/,
      loader: 'babel-loader'
    }]
  }
}
